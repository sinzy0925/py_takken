●還付の額
還付を受けることができる金額は、本店、支店、どこで取引しても、その業者が供託
している営業保証金の範囲内です。債権のすべてが弁済されるわけではありません。
例 本店＋支店3か所の宅建業者の場合
供託額 1,000万円＋500万円×3＝2,500万円
⇒還付を受けられる額も2,500万円以内

試験では電卓が使えません。日頃から、営業保証金の供託額や
報酬額（PART9で学習）は電卓を使わずに計算するようにし
しょう。計算ミスで得点を失うのは悔しいですからね

解答＆解
1 営業保証金の還付を受けるには、宅建業に関する取引によって生じた債権を持ってい
ることが必要です。建設工事を請け負った代金は、宅建業に関する取引によっ
て生じた債権ではありませんので、この建設業者は営業保証金から弁済を受けること
はできません。【×】

還付を受けられるのは、宅建業に関する取引によって生じた債権だけなんですね？

そうです。還付を受けられない事例として、電気工事を実施した電気業者、
内装工事を請け負った内装業者、宅建業者の依頼で広告を制作した広告業者
などが試験にも出ています

2 新たに支店を設置した場合は、本店の最寄りの供託所に政令で定める額の営業保証金
を供託しなければなりません。しかし、供託するだけでは、当該支店での事業を開始
することができません。さらに、供託した旨を免許権者に届出することで、事業を開
始できます。「①営業保証金の供託→②供託した旨を免許権者に届出→③事業開始」の
順番は、試験によく出ますので覚えておきましょう。【×】

3 営業保証金の還付により、供託している営業保証金の額に不足が生じたときは、宅建
業者は、追加で供託を行う必要があります。具体的には、免許権者から不足額を供託
すべき旨の通知があった日から2週間以内に、その不足額を供託しなければなりませ
ん。そして、不足額を供託した場合は、その日から2週間以内に供託した旨を免許権
者に届出しなければいけません。【○】

4 本店1,000万円＋（支店500万円×2）＝2,000万円が、供託すべき営業保証金の
合計額です。
【×】
正解：3

6 営業保証金の取戻し 重要度 ★★★
例題 2011年 問30
宅地建物取引業者A社（甲県知事免許）の営業保証金に関する次の記述のうち、宅地
建物取引業法の規定によれば、正しいものはどれか。
1 A社は、甲県の区域内に新たに支店を設置し宅地建物取引業を営もうとする場合、
甲県知事にその旨の届出を行うことにより事業を開始することができるが、当該支店
を設置してから3月以内に、営業保証金を供託した旨を甲県知事に届け出なければなら
ない。
2 甲県知事は、A社が宅地建物取引業の免許を受けた日から3月以内に営業保証金を
供託した旨の届出をしないときは、その届出をすべき旨の催告をしなければならず、
その催告が到達した日から1月以内にA社が届出をしないときは、A社の免許を取り
消すことができる。
3 A社は、宅地建物取引業の廃業により営業保証金を取り戻すときは、営業保証金の
還付を請求する権利を有する者（以下この間において「還付請求権者」という。）に
対して公告しなければならないが、支店の廃止により営業保証金を取り戻すときは、
還付請求権者に対して公告する必要はない。
4 A社は、宅地建物取引業の廃業によりその免許が効力を失い、その後に自らを売主
とする取引が結了した場合、廃業の日から10年経過していれば、還付請求権者に対
して公告することなく営業保証金を取り戻すことができる。

解法
営業保証金を取戻しできる事由と方法がよく出題されるので覚えておきましょう。

| 取戻し事由 | 取戻し公告の要否・方法 |
| :--- | :--- |
| ・事務所を一部廃止したため、供託している金額が政令で | 6か月以上の期間を定めて、一定期間内 |
| 定める金額より多くなった（廃止した分を一部取戻し） | に申し出るべき旨を公告しなければなら |
| ・免許の有効期間が満了した | ない |
| ・廃業・破産等の届出により免許が失効した | →公告後、遅滞なく、公告した旨を免許 |
| ・免許取消処分を受けた | 権者に届出 |
| ・有価証券を含んだ営業保証金を供託している場合で、主 | 公告不要 |
| たる事務所移転のため、移転先の供託所に営業保証金を | →ただちに供託金を取り戻すことが |
| 供託しなおした（二重供託） | できる |
| ・保証協会の社員となり、営業保証金の供託不要となった | |
| ・取戻し事由が発生したときから10年が経過した | |

解答＆解説
1 事業開始前に、供託した旨を甲県知事に届け出なくてはなりません。【×】
2 重要ポイントは「3か月以内に届出」「催告後1月」「免許を取り消すことができる（任
意）」です。「取り消さなければならない」ではありません。【○】
3 支店廃止が取戻し事由の場合は、6か月以上の期間を定めた公告が必要です。【×】
4 取引が結了した時が取戻し事由の発生時となり、そこから10年を経過すれば公告せ
ずに取戻しができます。【×】
正解：2

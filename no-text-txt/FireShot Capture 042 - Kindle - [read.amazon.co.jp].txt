成年者と同一の行為能力を有する未成年者は、法定代理人が欠格者であっても、本人
が欠格事由に該当しなければ、免許を受けることができます。
営業に関して成年者と同一の行為能力を「有しない未成年者」「有する未成年者」「法
定代理人」の3者を絡めたひっかけ問題が出やすいので、気をつけましょう。

| 法定代理人が欠格者の場合は？ | |
| :--- | :--- |
| 法定代理人から営業の許可を受けた未成年者 | 〇 免許を受けられる |
| 営業に関し成年者と同一の効力を有しない未成年者で法定代理人等に問題がある未成年者 | ✕ 免許を受けられない |

例題2～4で問われた欠格事由のほかに、
①免許申請前5年以内に宅建業に関し不正または著しく不当な行為をした者
②宅建業に関し不正または不誠実な行為をするおそれが明らかな者
③心身の故障により宅建業を適正に営むことができない者
も欠格者になることを覚えておきましょう（26、27ページの表⑥⑦⑧）。

解答＆解説
1 A社のように、正当な事由なく合併により消滅した場合、聴聞の期日及び場所の公示
日前60日以内に役員であった者は、合併消滅の日から5年を経過しなければ免許を
受けることができません（26ページの表②③）。 【〇】

免許の取消処分の聴聞の期日および場所の公示日前60日以内
に役員だった者は、消滅または届出の日から5年を経過しない
うちは欠格者です

2 欠格者が政令で定める使用人に就任した場合、その法人は免許を受けることができま
せん。このケースでは、Dの執行猶予期間が満了していないので、E社は免許を受け
られません（26、27ページの表⑤⑩）。 【〇】
3 営業に関し成年者と同一の行為能力を有しない未成年者である場合、本人FとFの法
定代理人であるGの両方について、欠格事由に該当するかが考慮されます。Gは背任
罪で罰金刑に処せられているので、その刑の執行が終わった日から5年を経過してい
なければ、Fは免許を受けられません（27ページの表①）。 【〇】
4 H社の役員であるIが暴力団員であることは、欠格事由に該当します。しかし、H社
の免許が取り消された後、Iが退任すればH社は免許を受けることができます（26、
27ページの表④⑩）。 【✕】

5 変更の届出 重要度 ★★★
例題 2009年 問28改
次の記述のうち、宅地建物取引業法（以下この問において「法」という。）の規定に
よれば、正しいものはどれか。

1 法人である宅地建物取引業者A（甲県知事免許）は、役員の住所について変更があった
場合、その日から30日以内に、その旨を甲県知事に届け出なければならない。
2 法人である宅地建物取引業者B（乙県知事免許）が合併により消滅した場合、Bを
代表する役員であった者は、その日から30日以内に、その旨を乙県知事に届け出な
ければならない。
3 宅地建物取引業者C（国土交通大臣免許）は、法第50条第2項の規定により法第
31条の3第1項の国土交通省令で定める場所について届出をする場合、国土交通大
臣への届出は、当該場所の所在地を管轄する都道府県知事を経由して行わなければな
らない。
4 宅地建物取引業者D（丙県知事免許）は、建設業の許可を受けて新たに建設業を営
むこととなった場合、Dは当該許可を受けた日から30日以内に、その旨を丙県知事
に届け出なければならない。

解法
免許権者（都道府県知事など）が宅建業者に免許を交付するときに、宅地建物取引業
者名簿に下記の事項を登載します。この登載事項の②～⑤に変更があった場合は、宅建
業者は30日以内に免許権者に届け出なければいけません。

| 宅地建物取引業者名簿の登載事項 | ②～⑤は変更時の届出が必要！ |
| :--- | :--- |
| ① 免許証番号・免許年月日 | ③④は旧姓併記申請可能！ |
| ② 商号・名称 | |
| ③ 法人の場合、役員（非常勤取締役、監査役含む）の氏名・政令で定める使用人の氏名 | |
| ④ 個人の場合、その者の氏名・政令で定める使用人の氏名 | |
| ⑤ 事務所の名称・所在地 | |
| ⑥ 指示または業務停止の処分を受けているときはその年月日と内容 | |
| ⑦ 宅建業以外の事業を行っているときは、その事業の種類 | |
| ⑧ 取引一任代理等の認可を受けているときは、その旨及び認可の年月日 | |

※事務所ごとに置かれる成年者である専任の宅建士の氏名は、変更の届出は必要（旧姓併記申請可）
役員、政令で定める使用人、専任の宅建士の住所や本籍地が変わっても変更
の届出は不要です。宅建業者名簿は公開されているので、そもそも個人の住
所は載せていません

解答＆解説
1 役員の氏名は変更の届出が必要ですが、住所の変更の届出は必要ありません。 【✕】
2 合併で消滅した場合は消滅会社の代表役員であった者が届出します（36ページ）。【〇】
3 国土交通大臣免許の場合、都道府県知事を経由せず、直接、国土交通大臣（各地方整
備局等）に届出をします。 【✕】
4 宅建業以外の兼業をする場合、変更の届出は不要です。 【✕】
正解：2

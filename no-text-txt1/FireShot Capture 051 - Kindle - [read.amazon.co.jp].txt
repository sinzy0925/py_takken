登録の移転の申請は、移転先の都道府県知事に対して、現に登録している都道府県知事を経由して行います。
また、事務の禁止期間中は、登録の移転はできません。
たとえば、会社が東京から大阪に移転したときには、登録の移転の申請は、どのように行うのでしょうか？

例）東京都登録の宅建士
　　現在登録している都道府県知事を経由して
　　移転先の都道府県知事に対して行う
（図省略）
会社が東京から大阪へ移転

また、登録の移転をすると、現在の宅建士証は失効するので、登録の移転申請とともに、宅建士証の交付申請も行う必要があります。
登録の移転後、新宅建士証は旧宅建士証と引換えに交付されます。この場合、知事指定の講習は受ける必要がありません。
また、新宅建士証の有効期間は、旧宅建士証の有効期間の残存期間となります。
宅建士の住所変更は、変更の登録、登録の移転、宅建士証の交付が関係します

解答＆解説
ア 宅建試験は未成年者でも受験することができます。また、未成年者でも法定代理人から同意を得て、営業に関し成年者と同一の行為能力を有する未成年者であれば登録できます。【×】
イ 宅建士の登録は従業地の都道府県知事の登録でなくてもよく、登録の移転をするとしても義務ではありません。乙県に移転しなければならないというのが誤りです。【×】
ウ 事務の禁止の処分を受けた場合、事務禁止期間が満了するまでは、登録の移転をすることができません。【○】
エ 登録の移転とともに宅建士証の交付申請をした場合、新宅建士証の有効期間は戊県で交付された宅地建物取引士証の有効期間が経過するまでの期間、つまり、旧宅建士証の有効期間の残存期間となります。【○】
変更の登録と登録の移転が出てきたら、ひっかからないように警戒してくださいね！
正解：2（二つ）

6 死亡等の届出　　　重要度 ★★☆
例題　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　　1991年 問38改
宅地建物取引業上の届出に関する次の記述のうち、誤っているものはどれか。
1 国土交通大臣の免許を受けている宅地建物取引業者A社が新たに政令で定める使用人を設置した場合、A社は、その日から30日以内に、その旨を国土交通大臣に届け出なければならない。
2 甲県知事の免許を受けている宅地建物取引業者B社が建設業の許可を取得して建設業を営むこととなった場合、B社は、その日から30日以内にその旨を甲県知事に届け出なければならない。
3 乙県知事の登録を受けている宅地建物取引士Cが心身の故障により宅地建物取引士の事務を適正に行うことができなくなった場合、Cまたはその法定代理人若しくは同居の親族は、その日から30日以内にその旨を乙県知事に届け出なければならない。
4 丙県知事の登録を受けている宅地建物取引士Eが死亡した場合、その相続人Fは、その事実を知った日から30日以内にその旨を丙県知事に届け出なければならない。

解法
宅建士登録を受けている者が死亡した場合などに、登録先の都道府県知事へ行う届出についての問題です。
①どんなときに、②誰が、③いつまでに届け出るのかがポイントです。たとえば、①死亡した場合、②相続人が、③その事実を知った日から30日以内に届け出ます。下の表の2〜7は、免許の欠格事由に該当することもポイントです。

| 届出事由 | 届出期限 | 届出義務者 |
| :--- | :--- | :--- |
| 1 死亡した | その事実を知った日から30日以内 | 相続人 |
| 2 心身の故障がある一定の者になった | | 本人・法定代理人、同居の親族 |
| 3 破産者になった | | |
| 4 成年者と同一の行為能力を有しない未成年者になった | その日から30日以内 | 本人 |
| 5 一定の事由により免許を取り消された | | |
| 6 一定の罪により罰金刑、または拘禁刑以上に処せられた ★法改正 | | |
| 7 暴力団員等になった | | |

解答＆解説
1 政令で定める使用人の氏名は、宅建業者名簿の登記事項なので30日以内に国土交通大臣に届け出なければなりません。【○】
2 宅建業以外の事業の種類については、業者名簿に記載します。ですが、変更の届出の対象項目ではありません。【×】
3 本人・法定代理人・同居の親族が、その日から30日以内に乙県知事に届出をします。【○】
4 相続人が、その事実を知った日から30日以内に丙県知事に届出をします。【○】
正解：2

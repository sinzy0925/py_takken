・賃貸人に無断の転貸は？
賃借人が賃借権の譲渡や転貸をするときは、賃貸人の承諾が必要ですが、賃借人が無断で譲渡、転貸をした場合は、原則として賃貸人は契約を解除することができます。ただし、賃貸人に対する背信的行為と認めるに足りない特段の事情がある場合は、賃貸人は契約を解除することはできません。

まとめ④ 対抗要件 [例題4]
・賃借権の対抗要件
不動産の賃借権の対抗要件は、賃借権の登記です。
たとえば、賃借権の対抗要件を備えている土地が第三者に譲渡された場合でも、賃借人は土地を明け渡す必要はありません。
（左図：A 賃貸人、C 新所有者、B 賃借人。賃借権の登記があれば出ていかなくてもよい）
（右図：A 賃貸人、B 賃借人、C 賃借人。賃借権の二重設定、賃借権登記を早く備えたほうが勝ち）

----------

・賃貸人の地位の移転
〈賃借人が対抗要件を備えている場合〉
一定の合意がある場合を除き、賃貸不動産の所有権が移転されれば、賃借人の地位も移転します。

〈賃借人が対抗要件を備えていない場合〉
譲渡人と譲受人との合意があれば賃借人の承諾なしに、移転させることができます。賃料請求など賃貸人の地位を賃借人に主張するには、新所有者は所有権移転登記を備える必要があります。
（図：旧賃貸人（譲渡人）、新賃貸人（譲受人・新所有者）、賃借人）

・賃借人・転借人の用法違反
賃借人の用法違反により賃貸物に損害が生じた場合において、賃貸人が損害賠償請求をするときは、賃貸物の返還を受けた時から1年以内しなければなりません。
なお、賃貸人は転借人の用法違反で生じた損害の賠償でも、賃借人に対して請求することができます。

まとめ⑤ 賃貸借と使用貸借 [例題5]
賃貸借　賃料を対価に物の貸し借りをすること
使用貸借　無償で物の貸し借りをすること

・賃貸借と使用貸借の違い
| | 賃貸借 | 使用貸借 |
| :--- | :--- | :--- |
| 解除 | できない（例外あり） | 書面で契約しているときは解除できない |
| 転貸 | 賃貸人の承諾を得ればできる | 貸主の承諾を得ればできる |
| 中途解約 | 期間の定めがなければいつでも解約できる<br>期間の定めのある場合は、解約権留保特約があるときに限りできる | 借主はいつでも契約の解除ができる<br>貸主からは契約に期間や目的の定めがある場合は、原則として中途解約できない<br>期間と目的の定めがない場合はいつでも契約の解除ができる |
| 賃貸人・貸主の損害賠償請求 | 目的物の返還を受けたときから1年以内に請求 | |

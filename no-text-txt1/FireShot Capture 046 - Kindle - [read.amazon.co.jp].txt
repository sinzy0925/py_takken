● 登録の移転のポイント
・登録の移転をすると、現在の宅建士証（旧宅建士証）は失効する
⇒登録の移転申請とともに宅建士証の交付申請を行う
・新宅建士証は、旧宅建士証と引き換えに交付される
・新宅建士証の有効期間は、旧宅建士証の残存期間となる（新たに5年ではない）
・登録の移転申請は、現に登録している都道府県知事を経由して行う
・事務の禁止処分の期間中は、登録の移転はできない

まとめ ⑤ 死亡等の届出 【例題 6】
宅建士の登録を受けている者が死亡したり、破産したりした場合には、その旨を登録先の都道府県知事に届出しなければなりません。
この届出を覚えるときには、
「どんなとき（届出事由）」「いつまでに（届出期限）」「誰が（届出義務者）」の3つに注意しながら、正確に覚えましょう。

| | 届出事由 | 届出期限 | 届出義務者 |
| :--- | :--- | :--- | :--- |
| 1 | 死亡した | その事実を知った日から30日以内 | 相続人 |
| 2 | 心身の故障がある一定の者になった | その日から30日以内 | 本人・法定代理人・同居の親族 |
| 3 | 破産者になった | | |
| 4 | 成年者と同一の行為能力を有しない未成年者になった | | |
| 5 | 一定の事由により免許を取り消された | | |
| 6 | 一定の罪により罰金刑、または拘禁刑以上に処せられた ★法改正 | | |
| 7 | 暴力団員等になった | | |

宅建業者の廃業等の届出と混同してしまいそうです……
宅建業者が破産したときは、破産管財人が届出義務者でしたが、宅建士が破産した場合は本人ですね。こうした違いに注意しましょう！

上記の表以外にも、
・本人から登録消除の申請があったとき
・死亡の事実が判明したとき
・試験の合格が取り消されたとき
には、都道府県知事は登録の消除をしなければなりません。

まとめ ⑥ 宅地建物取引士証 【例題 7】
宅建士資格の登録を受けた後に、宅地建物取引士証（以下、宅建士証）の交付を申請することができます。交付の申請は、登録をしている都道府県知事に対して行います。宅建士証の有効期間は5年です。
試験では、宅建士証に関する問題で、ひっかけ問題が作られやすいポイントが2つあります。

ひっかけポイント 1 宅建士証の交付を受けるときに、法定講習が必要か否か？
〈原則〉・交付申請前6か月以内に行われる、登録している都道府県知事が指定する講習（法定講習）の受講が必要
〈例外〉・宅建士試験合格から1年以内に宅建士証の交付申請をする場合
　　　・登録の移転の申請とともに、移転先の都道府県知事から宅建士証の交付を受ける場合
上記のように、法定講習を受けなくてもよい〈例外〉があることをしっかり覚えておきましょう

ひっかけポイント 2 交付時に受講が必要なのは、法定講習？ 登録実務講習？
宅建士証の交付を申請するときに必要な法定講習と、宅建士資格に登録する際の登録実務講習は別物ですが、これらを混同させるような問題が出ています。うっかりミスに気をつけてくださいね。

・宅地建物取引士証の重要ポイント
宅建士証の交付申請のほかに、宅建士証の書換え交付、宅建士証の返納義務や提出義務などについても、ひと通り覚えておきましょう。
下の表にまとめましたので、赤字のところは確実に覚えてください。まぎらわしいものほど出題頻度が高いです。
この後の例題を解きながら、それぞれ詳しく解説していきます。

| こんなときは？（発生事由） | やるべきこと（必要な申請や手続） |
| :--- | :--- |
| 氏名や住所に変更あり | 書換え交付の申請（変更の登録もあわせて行う） |
| 宅建士証を失くした、破損した | 再交付申請（見つけたらすみやかに旧宅建士証を返納） |
| 登録が消除された | すみやかに交付を受けた都道府県知事に返納 |
| 宅建士証が失効した | すみやかに交付を受けた都道府県知事に返納 |
| 事務の禁止処分を受けた | すみやかに交付を受けた都道府県知事に宅建士証を提出 ※事務の禁止期間が満了し、提出者が返還の請求をすればただちに返還される |

なお、上記の宅建士証の返納義務、提出義務に違反をした宅建士は、10万円以下の過料が科されます。

まとめ① 抵当権と不動産質権 [例題1]

・担保物権 債権者が債権を確実に回収するために備えておく権利
抵当権や質権は、契約によって成立する約定担保物権です。
・抵当権の目的物 不動産、地上権、永小作権
抵当権によって担保される被担保債権の範囲は、原則として元本のほか、利息その他の定期金などにつき、最後の2年分に限られています。ただし、後順位抵当権者などの利害関係者がいないときは、2年分に限定されません。
・抵当権の設定
抵当権の設定契約は、諾成契約（意思表示の合致によるもの）であり、抵当権者と抵当権設定者の合意で行われます。
・抵当権の対抗要件
他の債権者に抵当権を対抗したり、抵当権の目的物を買った第三者に対抗するには、登記が必要です。
・抵当権と質権

| | 抵当権 | 質権 |
| :--- | :--- | :--- |
| 被担保債権の利息 | 被担保債権の利息のうち、満期となった最後の2年分についてのみ担保される | 担保されない |
| 存続期間 | 制限なし | 10年を超えることはできない |
| 発生要件 | 諾成契約のため、目的物の引渡しは効力の発生要件ではない | 要物契約のため、目的物の引渡しが効力の発生要件となる |
| 対抗要件 | 登記 | 登記 |

まとめ② 物上代位・妨害排除請求 [例題2]

・物上代位 売買代金や賃料、抵当権の目的物の火災保険金などに抵当権の効力がおよぶこと
たとえば、抵当権の目的物である建物が火事で焼失した場合、火災保険金請求権や損害賠償請求権に対し、抵当権の効力を主張できます。
⇒目的物の所有者が受け取る保険金等を差押えることで、債権を回収

・物上代位のポイント
・抵当権者が物上代位するには、抵当権設定者（抵当不動産の所有者）がその保険金等を受領する前に差押えることが必要です。
・抵当権を設定していない土地上の建物が消失しても、建物の保険金を請求することはできません。

・抵当権の効力

| 対象 | 効力 |
| :--- | :--- |
| 付加一体物 | 増築部分や雨戸など、抵当不動産とそれに付加して一体となった物には、付加されたのが抵当権の設定の前後を問わず、原則として効力が及ぶ |
| 従物、従たる権利 | ・土地上に設置されている動かすことができる庭石や石灯籠など、抵当権設定当時に存在した従物については、当事者はそれも抵当権の対象であると考えているので、原則として効力が及ぶ<br>・抵当権設定後の従物には効力は及ばない<br>・土地を借りてその土地の上に建物を建てていた場合、建物に抵当権が設定されれば、設定当時からあった借地権などの土地利用権に対しても効力が及ぶ |
| 抵当不動産の果実 | 被担保債権に債務不履行があった場合、不履行後に生じた抵当不動産の果実にも効力が及ぶ |

・妨害排除請求
抵当不動産を不法に占有している第三者や、通常の利用方法を逸脱して、目的物を損傷する抵当権設定者に対して、抵当権者は妨害排除請求ができます。
・抵当権の順位変更
抵当権の順位は登記の順番ですが、各抵当権者の合意があれば、順位変更の登記をすることによって変更できます。変更には利害関係者の承諾が必要ですが、債務者の同意や承諾は不要です。

たとえばA銀行が抵当権を設定して、その登記をする前に、B銀行が抵当権を設定して登記をしたら、B銀行が第1順位の抵当権者になります

まとめ① 業務上の規制 【例題1】【例題2】【例題3】

・宅建業者が業務を行う上で常に行うべきこと
①誠実に業務を行う
②従業者に対しての教育
③守秘義務
・業務における禁止事項
①不当な履行遅延
②重要な事項の不告知、不実告知
③誤認させるような断定的判断等の提供
④手付貸与
・契約の締結の勧誘に先立って伝えるべきこと
①宅建業者の商号・名称
②勧誘を行う者の氏名
③契約の締結について勧誘をする目的である旨

まとめ② 監督処分・罰則 【例題4】【例題5】【例題6】【例題7】

・宅建業者に対する監督処分ができる処分権者
| | 免許権者 | 業務地管轄の知事 |
| :--- | :---: | :---: |
| 指示処分 | 〇 | 〇 |
| 業務停止処分 | 〇 | 〇 |
| 免許取消処分 | 〇 | × |

・宅建士に対する監督処分ができる処分権者
| | 登録している知事 | 行為地管轄の知事 |
| :--- | :---: | :---: |
| 指示処分 | 〇 | 〇 |
| 事務禁止処分 | 〇 | 〇 |
| 登録消除処分 | 〇 | × |

免許取消処分は免許権
者のみ、登録消除処分
は登録をしている都道
府県知事のみできます

宅建士が都道府県知事から指示処分を受けた場合において、宅建業者の責めに帰
すべき理由があるときは、免許権者は宅建業者に対して指示処分ができます。

・監督処分の手続きの流れ
[処分権者]
↓通知・公示
[公開の聴聞]
↓
[処分]
↓
[公告]
↓
[通知等]

国土交通大臣は、一定の規定違反を理由に
宅建業者に監督処分を行うときは
あらかじめ内閣総理大臣に協議しなければならない

国土交通大臣または都道府県知事は
宅建業者に対する監督処分や
宅建士に対する監督処分を行おうとするときは
公開の聴聞をしなければならない

業務停止処分または免許取消処分を
国土交通大臣が処分した場合は、
その旨を官報にて公告
都道府県知事の処分の場合は、
公報またはウェブサイトへの掲載等で
公告しなければならない

・監督処分の公告

| | 宅建業者 | 宅建士 |
| :--- | :---: | :---: |
| 指示処分 | × | × |
| 業務停止処分/事務禁止処分 | 〇 | × |
| 免許取消処分/登録消除処分 | 〇 | × |

〇：公告を行う ×：公告を行わない

・宅建士に対する罰則

10万円以下の過料
宅建士証の返納義務に違反した宅建士
宅建士証の提出義務に違反した宅建士
重要事項の説明時に、宅建士証を提示しなかった宅建士

まとめ③ 住宅瑕疵担保履行法 【例題8】【例題9】【例題10】

自ら売主となる宅建業者は、新築住宅の構造耐力上主要な部分（基礎や壁・柱な
ど）または雨水の浸入を防止する部分（屋根や外壁・雨水排水管等）の瑕疵（種類・
品質に関して契約の内容に適合しない状態）について、引渡しから10年、瑕疵担
保責任を負います。

[屋根]
[柱・壁・外壁]
[土台・基礎]

宅建業者自ら売主（媒介・代理は適用なし）
新築住宅
引渡しから10年責任

まとめ1 不動産取得税 【例題1】【例題2】【例題3】

●不動産取得税 土地や建物を取得したときにかかる税金。地方税

| 課税主体 | 不動産が所在する都道府県 |
| :--- | :--- |
| 納税義務者 | 不動産の所有権を取得した者 |
| 課税客体 | 不動産の所有権の取得に対して税金がかかる。有償、無償は関係ない<br>登記の有無も関係ない<br>売買、交換、贈与、新築、増築、改築（家屋の価値が増加した場合） |
| 非課税 | 取得者が国・地方公共団体であるとき<br>相続・法人の合併等によって不動産を取得したとき |
| 課税標準 | 固定資産課税台帳に登録されている価格<br>※新築家屋や増改築の場合は、都道府県知事が価格を決定する |
| 税率 | 標準税率は4%だが、土地・家屋（住宅）の場合は標準税率3%とする特例がある<br>住宅以外の家屋（店舗、事務所等）の場合は、標準税率のまま4% |
| 納付方法 | 普通徴収 |

課税主体は、不動産が所在する都道府県です。取得した人の住所
地ではありません

●不動産取得税の免税点（課税標準が以下の場合、課税されない）
①土地の取得 10万円未満
②建築による家屋の取得（新築・増改築） 1戸につき23万円未満
③その他の家屋の取得（売買・交換・贈与等） 1戸につき12万円未満

不動産取得税の免税点は、語呂合わせで覚えましょう！

●不動産取得税の特例
不動産取得税の課税標準は、原則、固定資産課税台帳に登録されている価格です
が、この課税標準を少なくする不動産取得税の特例があります。

| 種類 | 課税標準の特例 |
| :--- | :--- |
| 宅地 | 課税標準が固定資産課税台帳の登録価格の2分の1となる<br>不動産取得税＝固定資産課税台帳の登録価格×1/2×3% |
| 新築住宅 | 1,200万円の控除<br>不動産取得税<br>＝（固定資産課税台帳の登録価格－1,200万円）×3%<br>〈適用要件〉<br>床面積50㎡（戸建て以外の貸住宅は40㎡）以上240㎡以下 |
| 既存住宅 | ・個人も法人も新築住宅取得の場合は適用OK<br>・自己居住用も賃貸用住宅も適用OK<br>※既存住宅の場合は、個人が自己の居住の用に供するときのみで、<br>法人、賃貸用住宅は適用されない。控除額等も新築住宅と異なる |

●不動産の取得があるとみなされるのは？
新築家屋の場合、最初の使用または譲渡が行われた日に家屋の取得があったもの
とみなし、現に不動産の所有権を取得した者が納税します。
ただし、新築された日から6か月を経過しても使用または譲渡が行われないときは、
6か月を経過した日に取得があったものとみなします。
納税者の特例として、宅建業者等による建売販売などの場合、住宅を新築した日
から1年を経過しても使用または譲渡が行われないときは、1年を経過した日に取得
があったものとみなします。

まとめ2 固定資産税 【例題4】【例題5】【例題6】

●固定資産税 不動産を所有していることに対して課される税金。地方税

| 課税主体 | 固定資産が所在する市町村 |
| :--- | :--- |
| 納税義務者 | 原則、固定資産の所有者 ⇒賦課期日（1月1日現在）に登記簿または固定資産課税台<br>帳に所有者として登記されている者<br>例外①賃借が設定されている場合は賃借権者<br>②100年を超える存続期間の定めがある地上権者<br>③災害等により所有者が不明の場合、賦課期日における使用者<br>※固定資産が賃借されている場合、賃借権者に対して固定資産税を課すというルー<br>ルはない |
| 課税客体 | 土地、建物、償却資産に対して税金がかかる |
| 非課税 | 所有者が国・地方公共団体の場合 |
| 課税標準 | ・賦課期日（1月1日現在）での固定資産課税台帳に登録されている価格<br>・土地は更地価格。3年に一度、評価替えを行う |
| 税率 | 税率は1.4%（1.4/100） ※条例で1.4%と異なる税率を定めることができる |
| 納付方法 | 普通徴収<br>・納税通知書は遅くとも納期の10日前までに交付される<br>・納期は4月・7月・12月および2月において市町村の条例で定めるが、特別の事情<br>がある場合には異なる納期を定めることができる |

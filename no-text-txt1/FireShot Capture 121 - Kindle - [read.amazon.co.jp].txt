まとめ① 代理制度 [例題1]
代理 本人に代わって当事者となり、契約の締結等をすること
⇒代理の効果は本人に直接帰属する

〈代理行為をするのに必要な要件〉
①代理人に代理権があること
②本人の代理人であることを相手方に示していること（顕名（けんめい））
代理人が顕名をせずに契約した場合、代理人自身が契約したとみなされ、
本人に効果は帰属しません。
ただし、相手方が代理人であることを知っていたとき（悪意）や知ること
ができたとき（有過失）には、契約の効果は本人に帰属します。


----------



・制限行為能力者が本人の任意代理人になった場合
⇒制限行為能力者であることを理由に取り消すことはできない
・制限行為能力者が他の制限行為能力者の法定代理人となった場合
⇒制限行為能力者であることを理由に取り消すことができる
自己契約と双方代理
自己契約も双方代理も、原則、認められず、無権代理行為とみなされます。例外
として、本人があらかじめ許諾を与えている（あるいは追認した）場合や、登記申
請や単なる債務の履行の場合は認められます。

[図の説明]
本人A ⇔ B（Aの代理人であり相手方でもある）→ 自己契約
本人A ⇔ B ⇔ 相手方C（BはAの代理人でありCの代理人でもある）→ 双方代理

----------

まとめ② 復代理 [例題2]
復代理人 代理人に代わって、代理人がすべき仕事をする人
復代理人を選んでも、代理人の代理権は消滅しません。
復代理人の代理権の範囲は、代理人に与えられた権限の範囲内です。
復代理人の行った代理行為の効果は、本人に帰属します。
代理人の代理権が消滅すれば、復代理人の代理権も消滅します。

| | 任意代理 | 法定代理 |
| :--- | :--- | :--- |
| 復代理人の選任 | 原則：復代理人を選任することはできない<br>例外：①本人の許諾があるとき<br>　　　②やむを得ない事由があるとき | いつでも自由に<br>復代理人を選任することができる |
| 代理人の責任 | 本人と代理人との間の事務処理契約に<br>関する債務不履行として本人に対して<br>責任を負う | 原則：すべての責任を負う<br>例外：やむを得ない事由によって<br>復代理人を選任したときは<br>選任及び監督についてののみ<br>責任を負う |

代理と復代理の違いをしっかり覚えましょう

まとめ③ 代理権の消滅 [例題3]
代理人に与えられた代理権が消滅する場合があります。


----------



| | 本人 | 代理人 |
| :--- | :--- | :--- |
| 法定代理 | 死亡 | 死亡<br>破産手続開始の決定<br>後見開始の審判 |
| 任意代理 | 死亡<br>破産手続開始の決定 | |

・本人が死亡した場合
⇒代理人がその死亡につき善意無過失であったとしても、代理権は消滅
・代理権を持つ代理人が死亡した場合
⇒代理権の相続は生じない

まとめ④ 代理行為の瑕疵 [例題4]
・詐欺、強迫、錯誤、心理留保、虚偽表示によって代理人が契約した場合
⇒善意か悪意かなどについては、原則、代理人を基準に判定する
⇒代理行為の効果は本人に帰属するため、取消や無効を主張できるのは本人
・特定の法律行為の委託があるとき
⇒本人が悪意または有過失なら、代理人が善意であっても対抗できない
・代理人が詐欺や強迫を行った場合
⇒本人の善意・悪意に関係なく、相手方はその契約を取り消すことができる

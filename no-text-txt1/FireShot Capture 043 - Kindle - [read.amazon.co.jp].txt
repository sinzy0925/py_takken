6 廃業等の届出

重要度 ★★★

例題 1990年 問43
宅地建物取引業法に関する次の記述のうち、正しいものはどれか。

1 甲県知事の免許を受けている宅地建物取引業者Aが死亡した場合、Aの一般承継人は、Aが締結した契約に基づく取引を結了する目的の範囲内において、なお宅地建物取引業者とみなされる。
2 国土交通大臣の免許を受けている宅地建物取引業者B社と乙県知事の免許を受けている宅地建物取引業者C社が合併し、C社が消滅した場合、C社を代表する役員であった者は、その旨を国土交通大臣に届け出なければならない。
3 国土交通大臣の免許を受けている宅地建物取引業者D法人が設立許可の取消しにより解散した場合、D法人の清算人は、当該解散の日から60日以内に、その旨を国土交通大臣に届け出なければならない。
4 丙県知事の免許を受けている宅地建物取引業者Eが破産した場合、Eの免許は、当該破産手続開始の決定のときから、その効力を失う。

解法
宅地業者が廃業などで免許が失効する場合の届出は、「①どんなときに②誰が③いつまでに届出をすると④いつ免許が失効するか」がポイントです。たとえば、①死亡した場合に②相続人が③その事実を知った日から30日以内に届出をすると、④死亡したときにさかのぼって免許が失効します。死亡の場合だけが「その事実を知った日から30日以内に届出」で、その他は「その日から30日以内に届出」と区別してください。また、届出先が免許権者であることも覚えましょう。

| 届出事由 | 期限 | 届出義務者(個人) | 届出義務者(法人) | 免許失効の時期 |
| :--- | :--- | :--- | :--- | :--- |
| 死亡 | その事実を知った日から30日以内 | 相続人 | - | 死亡の時 |
| 破産手続開始の決定 | その日から30日以内 | 破産管財人 | 破産管財人 | 届出の時 |
| 廃業 | | 本人 | 代表役員 | |
| 法人の解散 | | - | 清算人 | |
| 法人の合併消滅 | | - | 消滅会社の代表役員 | 合併の時 |

宅地建物業者が死亡などにより免許が失効し、契約している相手方が損害を被るおそれのある場合は特別ルールがあります。「当該宅地業者が締結した契約に基づく取引を結了する目的の範囲内において、一般承継人（いわゆる相続人）などの一定の者は宅地業者とみなされる」というものです。ただし、免許の承継はできませんので、新たな取引を行うには、新たに免許を取得する必要があります。

解答＆解説
1 一般承継人は取引を結了する目的の範囲内で宅地業者とみなされます。【○】
2 C社の代表役員による乙県知事（C社の免許権者）への届出が必要です。【×】
3 60日以内ではなく、30日以内です。【×】
4 破産の場合は、破産管財人が免許権者に届出をした時点で失効となります。【×】
正解：1

7 免許換え

重要度 ★★★

例題 2020年12月 問29
次の記述のうち、宅地建物取引業法の規定によれば、正しいものはどれか。

1 宅地建物取引業者（甲県知事免許）が、乙県内に新たに事務所を設置して宅地建物取引業を営むため、国土交通大臣に免許換えの申請を行い、その免許を受けたときは、国土交通大臣から、免許換え前の免許（甲県知事）の有効期間が経過するまでの期間を有効期間とする免許証の交付を受けることとなる。
2 宅地建物取引士（甲県知事登録）が、乙県に所在する宅地建物取引業者の事務所の業務に従事することとなったため、乙県知事に登録の移転の申請とともに宅地建物取引士証の交付の申請をしたときは、乙県知事から、有効期間を5年とする宅地建物取引士証の交付を受けることとなる。
3 宅地建物取引士（甲県知事登録）が、乙県に所在する建物の売買に関する取引において宅地建物取引士として行う事務に関し不正な行為をし、乙県知事により事務禁止処分を受けたときは、宅地建物取引士証を甲県知事に提出しなければならない。
4 宅地建物取引業者（甲県知事免許）は、乙県内で一団の建物の分譲を行う案内所を設置し、当該案内所において建物の売買の契約を締結し、又は契約の申込みを受ける場合、国土交通大臣に免許換えの申請をしなければならない。

解法
宅建業者の免許権者は、次のルールで決まっていましたね。
・2つ以上の都道府県にまたがって事務所を設置する場合 ⇒ 国土交通大臣
・1つの都道府県内に事務所を設置する場合（複数の事務所でも可） ⇒ 都道府県知事

しかし、事務所の増減や移転などにより、このルールから外れてしまうことがあります。そこで必要になるのが免許換えです。免許換えの事由、申請先についてまとめました。

| 事由 | 免許換え |
| :--- | :--- |
| A・B県にまたがり事務所があったがA県のみの事務所になった | A県知事へ免許換えの申請 |
| A県のみの事務所だったがA県とB県の2か所に事務所を設けた | 国土交通大臣へ免許換えの申請 |
| A県からB県に移転した（事務所は1つ） | B県知事へ免許換えの申請 |

解答＆解説
1 免許換え後の有効期間は新しい免許が交付されてから5年です（39ページ）。【×】
2 宅建士証について問われています。この場合、新宅建士証の有効期間は登録移転前の宅建士証の残存期間となります（52ページ）。【×】
3 宅建士証について問われています。乙県知事により事務禁止処分を受けたときは、速やかに宅建士証を甲県知事に提出しなければなりません（55ページ）。【○】
4 乙県に案内所を設置しただけでは、免許換えの必要はありません。乙県に事務所を設置した場合は、国土交通大臣免許となり免許換えが必要になります。【×】
正解：3
